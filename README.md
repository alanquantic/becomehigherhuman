# Become Higher Human

Proyecto React con Vite listo para desplegar en Vercel.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Deploy en Vercel

Vercel detecta Vite automáticamente. Usa:

- Build Command: `npm run build`
- Output Directory: `dist`

Las imágenes principales están en `public/assets`.
